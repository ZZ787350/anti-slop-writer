# Vocabulary Banlist - English

> Words and phrases that signal AI-generated text

## Significance Puffers

Never use these words to describe importance:

- pivotal, crucial, vital, key (as adjective), significant, essential
- groundbreaking, remarkable, transformative, indelible, profound
- testament, enduring, lasting, deeply rooted, paramount
- indispensable, invaluable, quintessential

## Analytical Verbs

Replace with simpler alternatives:

| Banned | Use Instead |
|--------|-------------|
| underscore | show, prove |
| highlight (as "emphasize") | show, point out |
| showcase | show, display |
| foster | help, build |
| garner | get, receive |
| bolster | strengthen, support |
| delve | dig into, explore |
| embark | start, begin |
| leverage | use |
| facilitate | help, enable |
| utilize | use |
| encompass | include |
| cultivate | grow, build |
| elucidate | explain |
| illuminate | explain, show |
| navigate (figurative) | handle, manage |
| exemplify | show, illustrate |
| embody | represent |
| transcend | go beyond |
| harness | use |
| spearhead | lead |
| streamline | simplify |
| galvanize | motivate |

## Poetic Nouns

Avoid these metaphorical nouns:

- tapestry, landscape (figurative), realm, paradigm
- ecosystem (figurative), journey (figurative), nexus
- interplay, mosaic, fabric (of society), cornerstone
- beacon, pillar, catalyst, crucible, linchpin, hallmark
- confluence, odyssey, trajectory, underpinning

## Promotional Adjectives

Never use:

- vibrant, rich (figurative), comprehensive, robust
- seamless, innovative, dynamic, cutting-edge
- meticulous, intricate, nuanced, nestled, breathtaking
- renowned, diverse array, bustling, stunning
- multifaceted, holistic, overarching, compelling

## Puffery Adverbs

Avoid intensifiers that add nothing:

- seamlessly, meticulously, profoundly, intrinsically
- fundamentally, remarkably, notably, crucially
- undeniably, inherently, poignantly, relentlessly
- tirelessly, vividly

## Formal Connectives

Replace with simpler words:

| Formal | Simple |
|--------|--------|
| furthermore | also |
| moreover | also |
| consequently | so |
| accordingly | so |
| nonetheless | still |
| nevertheless | still |
| additionally | also |
| thus | so |
| hence | so |

## Opening/Closing Crutches

Never start or end with:

- "In today's world," "In today's fast-paced world"
- "In the ever-evolving landscape of"
- "In an era of/where"
- "As we navigate the complexities of"
- "In conclusion," "In summary," "Overall"
- "It is important to note that"
- "It's worth noting that"
- "At the end of the day"
- "Without further ado"
- "In a nutshell"
- "The bottom line is"
- "Last but not least"

## Fake Authenticity Signals

These are recognized AI tells:

- "But honestly?"
- "Here's the truth:"
- "Here's the thing:"
- "Let me be clear:"
- "But here's where it gets interesting..."
- "Think about it this way..."
- "Let me break this down..."

## Chat Artifacts

Never include these in output:

- "I hope this helps!"
- "Of course!"
- "Certainly!"
- "You're absolutely right!"
- "Would you like me to..."
- "Is there anything else..."
- "Let me know if..."
- "As an AI language model..."
- "I'd be happy to..."
- "Great question!"

## Vague Attribution Phrases

Never use without specific sources:

- "Experts argue"
- "Observers note"
- "Industry reports suggest"
- "According to some"
- "Many believe"
- "It is widely regarded"
- "Studies show" (without naming the study)
- "Research suggests" (without citing it)
